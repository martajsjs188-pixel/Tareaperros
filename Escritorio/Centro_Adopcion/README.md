# 🐶 Centro de Adopción

Aplicación web desarrollada con **Flask** y **MariaDB** para gestionar un centro de adopción de perros.  
Permite visualizar un catálogo de mascotas y registrar adopciones de manera sencilla.

---

## 📖 Descripción

Este proyecto simula el funcionamiento de un centro de adopción digital.  
Los usuarios pueden:

- Consultar un catálogo de perros disponibles.
- Adoptar una mascota (se actualiza el estado en la base de datos).
- Visualizar confirmaciones de adopción en la interfaz web.

---

## ✨ Características

- Catálogo dinámico conectado a MariaDB.
- Botón de adopción que actualiza el estado de cada perro.
- Plantillas HTML con Flask para la interfaz.
- Organización clara del código en rutas y plantillas.
